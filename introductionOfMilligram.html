Documentory Of Milligram

About Milligram

Milligram is a minimalist CSS framework designed to be lightweight and easy to use. It provides a clean and simple design, which is suitable for rapid prototyping and responsive design. Milligram is small (Gzipped) at around 2 kilobytes and contains only the essential CSS rules.

Features of Milligram

  • Light Weight
  • Provide simple and strong Grid Layout
  • Responsive Design

Grid System

Milligram provides a simple flexbox based grid system.

  • row
  • column
  • column-N
  • column-offset-N
                             
                                <div class="row">
                                    <div class="column column-50">Column 50%</div>
                                    <div class="column column-50">Column 50%</div>
                                </div>
                                <div class="row">
                                    <div class="column column-25">Column 25%</div>
                                    <div class="column column-75">Column 75%</div>
                                </div>

Typography :

Milligram provides ready-made classes for styling text and headings.

                            
                                <h1>Heading 1</h1>
                                <h2>Heading 2</h2>
                                <p>This is a paragraph.</p>
                                <blockquote>This is a blockquote.</blockquote>

Buttons

Classes can be used for different types of buttons.

  • button
  • button-outline
  • button-clear
  • button-primary
                            
                                <a class="button" href="#">Default Button</a>
                                <button class="button button-outline">Outline Button</button>

Forms

There are pre-defined styles for forms.

                            
                                <form>
                                        <fieldset>
                                            <label for="name">Name</label>
                                            <input type="text" id="name" placeholder="Enter your name">
                                            <label for="email">Email</label>
                                            <input type="email" id="email" placeholder="Enter your email">
                                            <label for="message">Message</label>
                                            <textarea id="message" placeholder="Enter your message"></textarea>
                                            <input class="button-primary" type="submit" value="Submit">
                                        </fieldset>
                                </form>

Tables

There is a default class for tables.

                            
                                <table>
                                    <thead>
                                        <tr>
                                            <th>Name</th>
                                            <th>Age</th>
                                            <th>Location</th>
                                        </tr>
                                    </thead>
                                    <tbody>
                                        <tr>
                                            <td>John</td>
                                            <td>28</td>
                                            <td>USA</td>
                                        </tr>
                                        <tr>
                                            <td>Jane</td>
                                            <td>25</td>
                                            <td>UK</td>
                                        </tr>
                                    </tbody>
                                </table>

Lists

                                        <ol>
